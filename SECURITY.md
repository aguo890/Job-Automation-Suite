## Reporting a Vulnerability

We take the security of our project seriously. 

If you discover a security vulnerability, we appreciate your help in disclosing it to us in a responsible manner. 
Please **do not** report security vulnerabilities through public GitHub issues.

Instead, please report them to: 
**compsci.aaron.guo@gmail.com**

You should receive a response within 48 hours. If for some reason you do not, please follow up via email to ensure we received your original message. We will keep you updated as we investigate the vulnerability and when a fix will be available.

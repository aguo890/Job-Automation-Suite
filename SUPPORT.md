# Support

Thank you for your interest in Job Automation Suite! We want to make sure you get the help you need. 

Please review the following options to determine the best place to ask your question or report your issue.

## 🐛 Bug Reports & 💡 Feature Requests
If you have found a reproducible bug or want to propose a new feature, please [open an issue](https://github.com/aguo890/Job-Automation-Suite/issues/new/choose) and use the appropriate template. 

## ❓ Questions & Help
If you have a question about how to use the project, or need help troubleshooting a non-bug issue:
1. First, check the `README.md` and `docs/` (if available) for relevant instructions.
2. If you still need help, **please use [GitHub Discussions](https://github.com/aguo890/Job-Automation-Suite/discussions)** (or the project's designated chat/Q&A forum) rather than opening a bug report. This helps keep the issue tracker focused on actionable development tasks while building an accessible knowledge base for everyone.

## 🔒 Security Vulnerabilities
If you believe you have found a security vulnerability, **do not open a public issue.**
Please refer to our [Security Policy](SECURITY.md) for instructions on how to report it securely.
